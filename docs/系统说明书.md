# 集团级经营管理+生产协同+任务督办+绩效考核一体化平台

## 系统说明书

**版本**：1.0  
**更新日期**：2025年3月

---

## 一、系统概述

### 1.1 建设目标

建设一套集团级经营管理+生产协同+任务督办+绩效考核一体化管理平台，实现全流程线上化、闭环化、数据化，打通各环节数据链路，提供统一工作入口。

### 1.2 覆盖对象

- 经营人员
- 生产调度
- 生产车间
- 质检部门
- 企管部门
- 集团管理层

### 1.3 核心参数

| 参数 | 说明 |
|------|------|
| 用户规模 | 约 200 人 |
| 组织模式 | 多法人单位，经营部门集团直管 |
| 部署方式 | 优先本地化部署，支持 Vercel 云部署 |
| 建设原则 | 稳定可靠、长期使用、移动端友好、成本可控 |

### 1.4 版本说明

| 版本 | 后端 | 数据库 | 说明 |
|------|------|--------|------|
| **当前 Demo** | TypeScript + Supabase | Supabase (PostgreSQL) | 为便于演示，使用 Supabase 一站式服务 |
| **计划正式版** | Java Spring | MySQL | 自建后端 + MySQL，满足企业级部署与集成需求 |

## 三、架构设计

### 3.1 技术架构

**说明**：当前为 Demo 版本，为便于演示使用 Supabase（PostgreSQL）。正式环境计划采用 Java Spring 后端 + MySQL 数据库。

#### 当前 Demo 架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户端 (Browser)                           │
│                    React 19 + Vite 6 + Tailwind 4                 │
└────────────────────────────┬────────────────────────────────────┘
                             │ HTTPS
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    前端应用 (SPA)                                  │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐  │
│  │ React Router │  │  Recharts   │  │  Supabase Client (TS)   │  │
│  │   (路由)     │  │  (图表)     │  │  (数据访问 - Demo)      │  │
│  └─────────────┘  └─────────────┘  └────────────┬──────────────┘  │
└─────────────────────────────────────────────────┼────────────────┘
                                                  │ REST/Realtime
                                                  ▼
┌─────────────────────────────────────────────────────────────────┐
│                    Supabase (BaaS)                                │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐               │
│  │  Postgres   │  │   Auth      │  │   RLS       │               │
│  │  (数据库)   │  │  (认证)     │  │  (行级安全)  │               │
│  └─────────────┘  └─────────────┘  └─────────────┘               │
└─────────────────────────────────────────────────────────────────┘
```

#### 计划正式架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    前端 (React + TypeScript)                      │
└────────────────────────────┬────────────────────────────────────┘
                             │ REST API
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    后端 (Java Spring)                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐               │
│  │  Controller │  │  Service    │  │  Repository │               │
│  └─────────────┘  └─────────────┘  └──────┬──────┘               │
└───────────────────────────────────────────┼──────────────────────┘
                                            │

```
